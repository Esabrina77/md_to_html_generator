# Bienvenue sur Mon SSG en Go
Ceci est la page principale de notre site, générée à partir d'un simple fichier Markdown.

Le générateur convertira :
* Le texte en **gras**.
* Les listes à puces.
* Les blocs de code comme `print("Hello, Go")`.

C'est un excellent projet pour apprendre les bases du Go et de la génération statique.
